# Building Godot Gaussian Splatting

Canonical published docs for build tasks live in:

- `docs/BUILDING.md` for building a module-enabled editor from this fork
- `docs/reference/build-test-ci.md` for test runners and CI command surfaces

Run build commands from repository root. The upstream Godot engine now lives at
repository root, and `modules/gaussian_splatting/` is the authoritative in-tree
module path.

If build/test commands in other docs differ, treat `docs/BUILDING.md` as the source of truth for base editor builds.
