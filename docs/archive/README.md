# Documentation Archive

This section stores point-in-time and historical materials that are useful for record-keeping but are not part of the canonical onboarding or day-to-day reference paths.

## Inclusion Rules

Archive a document when it is primarily one of:

- investigation log with date-specific context
- short-lived plan tied to a past execution window
- benchmark or audit snapshot tied to a specific run date

## Exclusion Rules

Do not archive pages that are canonical references for current behavior, setup, or contribution workflow.

## Usage Rules

- Do not link archived pages from primary quickstart/onboarding pages.
- Keep archive links only from index pages or historical notes.
- Preserve original dates in filenames when available.
